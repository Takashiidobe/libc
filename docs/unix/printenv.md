SYNOPSIS
========

```sh
printenv [OPTION]... [VARIABLE]...
```

DESCRIPTION
===========

Print values of the specified environment VARIABLE(s). If none is provided, print name and value pairs for all ENV_VARS.

```sh
-h, --help 
  display help and exit
-v, --version 
  output version information and exit
```
